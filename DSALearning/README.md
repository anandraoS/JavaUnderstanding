📊 COMPLEXITY CHEAT SHEET
========================

ARRAYS & STRINGS:
- Access: O(1)
- Search: O(n) linear, O(log n) binary
- Insert: O(n)
- Delete: O(n)

LINKED LISTS:
- Access: O(n)
- Search: O(n)
- Insert: O(1) if position known
- Delete: O(1) if position known

TREES (Balanced):
- Search: O(log n)
- Insert: O(log n)
- Delete: O(log n)
- Height: O(log n)

GRAPHS:
- DFS: O(V + E)
- BFS: O(V + E)
- Dijkstra: O((V + E) log V)
- Floyd-Warshall: O(V³)

SORTING:
- Quicksort: O(n log n) avg, O(n²) worst
- Mergesort: O(n log n) guaranteed, stable
- Heapsort: O(n log n) guaranteed
- Insertion: O(n) best, O(n²) avg/worst

HASH TABLE:
- Insert: O(1) avg, O(n) worst
- Search: O(1) avg, O(n) worst
- Delete: O(1) avg, O(n) worst
